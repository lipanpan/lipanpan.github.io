java 关机钩使用

发表于 2017-01-07   |   分类于 java

前言

在java中通过使用Runtime.getRuntime().addShutdownHook(Thread hook)方式，向JVM中注册一系列shutdownHook，当JVM关闭时，会一一调用注册的hook，通过利用这一特性，我们可以在钩子中做资源的回收和释放等操作。
