前言

以前经常看到一些开源软件的管理后台，都是基于浏览器弹框实现的认证登录；比如: haproxy管理后台，elastic-job管理后台等。在此梳理下其实现原理，进一步加深自己的理解。

Authorization

在http协议中请求头Authorization专门用于请求认证使用，用于透传凭证信息。Authorization有多种认证方式，最常见的一种就是Basic方式；通常开源软件管理后台也都是基于此方式实现的。
